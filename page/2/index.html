Dynamic Programming

动态规划的本质是递归，一切动态规划问题一定能写成递归的形式。递归的本质是原问题能拆解成子问题，子问题再推导出原问题。
最优子结构：用递归式处理掉后效性，使子问题不具有后效性。
重叠子问题：用空间换取时间，将中间结果记录下来，去除冗余。
问题共性： 套路：最大，最小，最优，最长，计数。
离散问题：用动态规划来解决，容易设计状态。
最优子结构：子问题的解能推导出原问题的解。N-1可以推导出N。

Tree

1. Tree Terminology

  • Root: node without parent (A)
  • Siblings(兄弟): nodes share the same parent
  • Internal node(内部结点): node with at least one child (A, B, C, F)
  • External node (leaf ): node without children (E, I, J, K, G, H, D)
  • Ancestors (祖先)of a node: parent, grandparent, grand-grandparent, etc.
  • Descendant(后裔) of a node: child, grandchild, grand-grandchild, etc.
