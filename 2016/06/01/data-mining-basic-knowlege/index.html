Data Mining Material

1.Introduction

  1. Q: why data mining?
    Answer: Data Rich, Information Poor

  2. Q: what is big data?
    Answer: volumn, velocity, varity, value.大量快速产生，多样化的数据需要花费很大的精力去处理以产生有用的信息和决策；数据量达到传统数据库所不能存储，处理和分析。

  3. Q:what's data?
    Answer: 数据是一条一条的信息，是一个属性的变量的定量或定性的表示。数据经常被视为从中可以抽取出知识和信息的最低级别的抽象。

  4. Data Types: continuous binray, discrete, string, symbolic.
    Storage: Physical, logical. Major issues: transformation, Errors and Corruption.

  5. Relational Database

  6. 第一范式：https://www.1keydata.com/database-normalization/first-normal-form-1nf.php; A database is in first normal form if it satisfies the following conditions: Contains only atomic values; There are no repeating groups. An atomic value is a value that cannot be divided. For example, in the table shown below, the values in the [Color] column in the first row can be divided into "red" and "green", hence [TABLE_PRODUCT] is not in 1NF. A repeating group means that a table contains two or more columns that are closely related. For example, a table that records data on a book and its author(s) with the following columns: [Book ID], [Author 1], [Author 2], [Author 3] is not in 1NF because [Author 1], [Author 2], and [Author 3] are all repeating the same attribute.

  7. 第二范式：https://www.1keydata.com/database-normalization/second-normal-form-2nf.php; A database is in second normal form if it satisfies the following conditions: It is in first normal form; All non-key attributes are fully functional dependent on the primary key. 第二范式在第一范式的基础上消除了非主属性对于码的部分函数依赖。如何判断一个表是否符合2NF?看数据表中是否存在非主属性对于码的部分函数依赖。3NF在2NF的基础之上，消除了非主属性对于码的传递函数依赖。BCNF在3NF的基础上消除了主属性对于码的部分与传递函数依赖。

  8. Data Warehouse Answer: 可操作的数据库，可以充分保持数据一致性和保证业务交易的速度，在数据仓库数据可以被很快的检索到。在数据仓库中存储公司的电子数据，以更好的利用这些数据仓库中的数据进行决策报告和分析。Subject-oriented, Time-variant, Non-volatile(挥发), Integrated.　数据仓库是一个业务智能工具。数据仓库是提取／转换,加载数据到一个仓库的工具。管理和检索元数据的工具。

  9. ETL: Extraction,Transformation,Loading.

  10. OLAP: Online Analytical Processing.　联机分析处理。

  11. To build a data warehouse?

  12. 数据挖掘是从大量的不完整和有噪声的数据中提取有意义的和隐藏的模式的自动化过程（并不是完全自动化的过程）。
